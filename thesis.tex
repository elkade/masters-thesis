\begin{abstract}
\paragraph{} The topic of our engineering thesis is to create a content management system designated to represent a faculty division. The main purpose of the system is to improve communication between the teaching staff and students and fulfill the roles of teaching service, information service and forum. The system should allow teachers to publish data in simple way and share it with the students. The forum module improves communication between students and allows them to speak there as soon as they receive the account.
The aim of the extensive administrative module is the efficient management of users and sub-pages. Thanks to the possibility to introduce new language versions, there are no obstacles for the system to be used by people using different languages.
We wanted the system to be intuitive and to enable content management also for people unfamiliar with web technologies. By design, system architecture provides the easy modification and expansion.
\paragraph{} As a result of our work a web application was created and it can be actually used by the faculty. The main part of the application was made with .NET \cite{dotnet}, using ASP.NET MVC application platform \cite{aspnetmvc} and the IIS server \cite{iis}. To a large extent, we used angularjs \cite{angular} library in order to create a visual layer. The database used in the system is based on Microsoft SQL Server \cite{sqlserver}.
A part of the written work are technical documentation and user manual for the application that have been created. The application code also contains a full XML documentation. This makes the opportunity to develop applications in the future by third parties.
